# Web research and freshness

Require live web research when named current products, providers, models, features, Stanford approval, data classification, service status, costs, benchmarks, capabilities, terms, policies, hosting options, or changed facts matter.

Source order: official Stanford sources for Stanford claims; vendor technical documentation/model cards; vendor security/privacy/terms/subprocessors; primary research/evaluations; credible independent reporting; stable background knowledge.

Freshness note: Research checked: date. Product/version examined: if relevant. Stanford source status checked: yes/no public source located/unavailable. Label confirmed, vendor-claimed, independently evaluated, inferred, and unknown.

Never infer absence from public Stanford materials means prohibited. Say: “I did not locate this product in the public Stanford materials I checked.”
