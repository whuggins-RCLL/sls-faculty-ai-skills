# System layer model

Core layers: human purpose and accountable owner; interface; workflow and instructions; institutional knowledge and retrieval; tools and connectors; identity, permissions, and access controls; data flow, storage, logs, retention, and deletion; evaluation and monitoring; model or models; hosting and computing infrastructure; optional adaptation or fine-tuning; foundation-model pretraining.

Text equivalent: an AI system is not just a model. It is a purpose-driven arrangement of people, interfaces, instructions, data sources, permissions, tools, logging, evaluation, models, infrastructure, and sometimes adaptation or training. Not every system has every layer.
