# Assistants, agents, tools, and connectors

A chatbot mostly exchanges messages. An assistant may add instructions, files, memory, retrieval, or workflow. An agent generally combines a model, instructions, tools, state, permissions, workflow, action rules, and human-confirmation points.

Do not anthropomorphize agents as employees or colleagues. Describe delegated software authority.

Action authority ladder: can answer; can retrieve; can recommend; can draft; can modify; can communicate; can publish; can approve; can delete. Each higher level changes risk, evidence needs, and oversight.
