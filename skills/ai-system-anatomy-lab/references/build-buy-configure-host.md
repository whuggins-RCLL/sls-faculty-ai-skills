# Build / buy / configure / host framework

Options: improve the non-AI process; configure an approved general-purpose service; create a faculty Skill or workflow; add controlled retrieval; build a limited app on an existing model; buy a domain-specific application; partner with internal/external technical group; host an open-weight model; adapt or fine-tune; conduct a research prototype; do not proceed.

Assess: problem fit, evidence, stakes, reversibility, verifiability, data sensitivity, access and permissions, expertise, maintenance, evaluation burden, human-review burden, accessibility, cost/opportunity cost, exit strategy.

Decision brief sections: problem; options; benefits; costs; dependencies; evidence needed; risks; reversibility; recommended next experiment; decisions reserved for authorized humans.
