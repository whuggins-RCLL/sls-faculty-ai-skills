# Faculty use cases

For each use case show problem, whether AI is necessary, relevant layers, required data, human judgment, verification, key risk, simpler alternative, safe first experiment.

Examples: source-grounded assistant for assigned readings; research-note organizer; current-awareness workflow; seminar-question generator; formative-feedback draft for faculty review; searchable public scholarship archive; public FAQ assistant; supplied-source comparer; research-assistant workflow organizer; structured AI-literacy exercise.

Do not recommend sensitive, restricted, student-record, confidential, privileged, or unpublished material unless the service and use have been confirmed appropriate. Prefer synthetic or public data for first experiments.
