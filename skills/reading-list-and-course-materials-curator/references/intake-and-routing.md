# Intake and Routing

## Acceptable inputs
Course title, description, syllabus, weekly topics, objectives, existing lists, prior syllabi, cases, statutes, regulations, articles, books, reports, news, podcasts, videos, websites, databases, library guides, faculty notes, reading limits, student level, credits, class duration, source balance preferences, access constraints, and notes about outdated or inaccessible materials.

## Paths
- Path A: Build a new reading list when topics or objectives exist but no complete list exists. Clarify scope, level, weeks, load, material types, current events, multimedia, and whether legal research is itself an objective.
- Path B: Revise an existing list. Diagnose outdated authorities, broken links, excessive volume, redundancy, missing primary sources, imbalance, unsupported objectives, inaccessible formats, paywalls, overreliance, sequencing problems, and assumed prior knowledge before recommending replacements.
- Path C: Update for currency. Search current primary and authoritative sources, distinguish foundational readings from updates, state search date, avoid replacing canonical materials only because they are older, and identify what became outdated and why.
- Path D: Targeted weekly module. Output only that module unless broader review is requested.
- Path E: Audit workload and balance. Check pages/time, difficulty, density, source count, primary/secondary mix, required/optional distinctions, context, credits, and design fit.
- Path F: Annotated list. Explain why each item is assigned and what students should do with it.
