# Intake and routing

Activate when faculty ask to create Socratic questions, prepare cold-call sequences, build follow-up questions, design hypotheticals, create factual variations, develop case-discussion questions, compare authorities through questioning, create policy questions, design branching responses, revise weak question sequences, or produce teaching notes for a hypothetical.

Do not activate merely for substantive legal answers, full syllabi, complete lesson plans without emphasis on questioning, exams or grading rubrics, student discipline, accommodations, or witness-examination strategy for a live matter.

Accept any combination of case name, uploaded opinion, excerpt, statute, syllabus, objectives, class notes, existing questions, hypotheticals, preparation questions, doctrine, time, student level, style, difficulty, misconceptions, cold-call practice, and preference for real, fictional, or hybrid materials.

## Paths

- Path A: case-focused sequence when a case or opinion is supplied or requested.
- Path B: doctrine-focused sequence when teaching rules across authorities.
- Path C: original hypothetical when a new fact pattern is requested.
- Path D: revision when existing questions or hypotheticals are supplied.
- Path E: branching follow-ups when faculty want response pathways.
- Path F: targeted component when only an opening, cold call, policy question, variation, synthesis question, preparation guide, or similar component is requested.

If essential information is missing, ask at most three high-value questions; otherwise state assumptions and continue.
