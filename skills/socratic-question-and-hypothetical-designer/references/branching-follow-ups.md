# Branching follow-ups

Use branches to help faculty respond constructively rather than script a cross-examination.

Format:

Primary question
├── Correct but incomplete answer
│   └── Ask for missing reasoning or authority; then transition.
├── Correct rule, weak application
│   └── Direct attention to the decisive fact; then synthesize.
├── Plausible alternative answer
│   └── Test implications or counterargument; then compare.
├── Mistaken answer
│   └── Ask a diagnostic question before correcting; then restate the point.
└── No response
    └── Offer a narrower entry question, text reference, pair consult, or handoff; then return with an invitation to revise.

Do not end a branch with embarrassment or silence. Each branch needs a correction, transition, or synthesis.
