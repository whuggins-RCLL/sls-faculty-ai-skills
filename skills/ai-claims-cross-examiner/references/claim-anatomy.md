# Claim anatomy

For substantial analyses, decompose the claim:

| Element | Question |
|---|---|
| Exact wording | What was actually said? |
| Claim source | Who or what made the claim? |
| Date | When was it made? |
| Claim type | Technical, empirical, legal, normative, predictive, or other? |
| Defined subject | Which model, product, system, or practice? |
| Task | What is it said to do? |
| Degree | Can, usually, reliably, always, better, safer, faster? |
| Conditions | Under which circumstances? |
| Comparator | Better than what? |
| Evidence standard | What would support the claim? |
| Scope | How far does the claim reach? |
| Consequence | What decision is the claim being used to justify? |

Do not invent missing elements. Mark them unknown or implicit.

## Scope funnel

Narrow from broad category to specific testable proposition:

1. Broad claim as stated
2. Strongest defensible version
3. Narrowest testable version
4. What would falsify or weaken each level
