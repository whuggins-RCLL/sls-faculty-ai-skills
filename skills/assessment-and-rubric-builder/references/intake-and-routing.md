# Intake and routing

Use the smallest useful intake. If the request is vague, ask for at most course, student level, primary objectives, grading weight, and AI posture; otherwise proceed with assumptions.

## Paths
- Path A: new assessment. Select format before drafting.
- Path B: revise existing assessment. Diagnose before rewriting.
- Path C: rubric only. Keep the assignment stable unless criteria reveal a material defect.
- Path D: stress-test. Identify ambiguity, loopholes, point issues, missing facts, unequal options, misunderstanding risks, AI vulnerabilities, and mismatched time pressure.
- Path E: formative assessment. Keep low-stakes checks proportional and feedback-rich.

## Confidential and restricted material warning
Before processing unreleased exams, restricted question banks, student records, identifiable student work, accommodation details, misconduct investigations, or confidential peer-review material, advise de-identification, synthetic samples, institutionally approved systems, minimal necessary disclosure, and faculty review before reuse. Do not assume uploading an exam to an AI platform is permitted.
