# Assessment selection

Choose formats by evidence sought, not by habit.

| Desired learning evidence | Possible assessment |
|---|---|
| Recall and recognition | Short answer or carefully designed multiple choice |
| Issue spotting | Fact-pattern essay |
| Rule application | Problem-based analysis |
| Research ability | Research memorandum or source trail |
| Written advocacy | Brief, motion, comment letter, or policy memo |
| Oral advocacy | Argument, presentation, or hearing simulation |
| Transactional judgment | Drafting or negotiation exercise |
| Professional judgment | Simulation with reflection |
| Synthesis and originality | Seminar paper or project |
| Collaboration | Structured group exercise with individual accountability |
| AI literacy | Verification, critique, comparison, or disclosure exercise |

Do not assume that a final exam is right merely because a course is doctrinal.
