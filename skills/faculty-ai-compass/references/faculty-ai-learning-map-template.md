## Required elements (validator checklist)

A complete Faculty AI Learning Map must include:

- Faculty purpose
- Existing strengths (specific)
- Current uncertainty (as questions)
- Problem or decision
- Misconceptions worth testing (when relevant)
- Recommended first destination
- Reason for the route
- Time and depth
- One immediate experiment
- Source or product freshness note when relevant
- Follow-up route (optional only)
- Reflection prompt

## Prohibited elements

- Numerical competency score
- Percentile
- Beginner or advanced ranking
- Remedial designation
- Personality profile
- Predicted job performance
- Stigmatizing label
