# Faculty AI Field Notebook

A reusable personal record owned by the faculty member. The Compass helps create entries; it does not centralize or score them.

## Entry template

See `field-notebook-template.md`.

## Usage

- Create an entry after a route recommendation, lab, or reflection
- Add new entries on return visits; do not overwrite prior history
- Use public, synthetic, or non-sensitive examples only

## Do not

- Create central performance tracking
- Infer private personal data
- Store sensitive information in the notebook
- Require completion of every Skill

## Connection to Learning Map

The Learning Map holds the current route plan. The Field Notebook holds dated observations and changed understanding. On return, read both and follow `return-and-resume-workflow.md`.
