# Controlled test patterns

Sixteen reusable guided lab patterns. Each follows the experiential learning cycle and uses safe public or synthetic materials by default.

## 1. Hallucinated authority lab

**Purpose:** Test whether the system invents or misstates legal authority.

**Materials:** Public law, clearly fictional authorities labeled FICTIONAL INSTRUCTIONAL MATERIAL, or faculty-supplied verified materials.

**Procedure:** Ask a question requiring citation; verify existence, identity, and support. Never present invented authority as real.

**Inspect:** Fabrication, misquote, wrong court, nonexistent case.

**Does not prove:** Universal hallucination rate across all legal tasks.

## 2. Citation support lab

**Purpose:** Determine whether cited sources support attributed propositions.

**Check:** Existence, identity, quoted language, proposition support, context, contrary material, current status when relevant.

**Does not prove:** Citator-level validation without an actual citator.

## 3. Missing-source lab

**Purpose:** Observe whether the system admits required information is unavailable.

**Setup:** Question whose answer is deliberately absent from supplied materials.

**Evaluate:** Acknowledges absence; cautious inference; fabricates answer; searches outside authorized source set.

## 4. Silent-omission lab

**Purpose:** Test for missing exceptions, counterarguments, procedural posture, minority views, limitations, adverse authority.

**Key insight:** No obvious false statement may still be materially incomplete. Use reference checklist.

## 5. Sycophancy lab

**Purpose:** Test whether conclusion changes when user reveals a preference.

**Procedure:** Neutral prompt vs preference-signaled variant. Compare legal reasoning and evidence. Do not frame system as malicious.

## 6. Framing and anchoring lab

**Purpose:** Compare equivalent but differently framed questions.

**Record:** Which changes are justified by framing and which are not.

## 7. Variability lab

**Purpose:** Run materially identical requests multiple times.

**Compare:** Conclusion, sources, confidence, omissions, structure, recommendations. Do not treat stylistic variation as substantive failure.

## 8. Prompt-sensitivity lab

**Purpose:** Alter one meaningful prompt element at a time.

**Avoid:** Comparing completely different requests and labeling difference "instability."

## 9. Context-collision lab

**Purpose:** Test conflicting information or instructions in user instructions, uploaded files, retrieved pages, embedded text, or earlier conversation.

**Safeguard:** Harmless synthetic embedded instructions only. No harmful prompt-injection content.

## 10. Retrieval lab

**Purpose:** Separate failure to retrieve, failure to interpret, failure to cite, failure to synthesize, failure to identify missing coverage.

**Include:** Answer absent from source collection.

## 11. Confidence-calibration lab

**Purpose:** Compare certainty of language with quality of available evidence.

**Note:** Verbal confidence does not map reliably to internal probability.

## 12. Tool-awareness lab

**Purpose:** Test whether assistant accurately states what tools, files, connectors, or actions it actually used.

**Inspect:** Logs, citations, reproducibility—not self-report alone.

## 13. Long-context and document lab

**Purpose:** Test across supplied collection with relevant sections, distractors, conflicting passages, similar terminology, intentionally absent answer.

**Does not prove:** Complete context capacity of the system.

## 14. Human-review lab

**Purpose:** Measure whether qualified reviewer can identify planted or known errors within realistic time.

**Reveal:** "Human in the loop" is a workflow claim, not a complete safeguard.

## 15. Agent-permission tabletop lab

**Purpose:** Map what could happen if an agent can read, retrieve, draft, modify, send, publish, approve, delete.

**Method:** Simulation only unless explicitly authorized tools are available.

## 16. Bias and equity lab

**Purpose:** Test whether materially equivalent examples receive uneven treatment.

**Require:** Clear hypothesis, controlled variations, multiple examples, caution about generalization, review of confounding variables. No claims about protected groups from a single anecdote.
