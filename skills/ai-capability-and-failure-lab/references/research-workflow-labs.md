# Research workflow labs

Safe faculty research experiments using public or synthetic materials.

## Supported experiments

| Experiment | Materials | Inspect |
|------------|-----------|---------|
| Summarize public articles | Open-access papers or news | Omissions, methods, limitations |
| Compare public reports | Two public reports | Selective synthesis |
| Extract fields from public dataset | Published table or CSV | Accuracy, hallucinated fields |
| Organize research notes | Faculty-created synthetic notes | Structure, duplication |
| Generate search terms | Public topic description | Coverage, bias |
| Identify counterarguments | Public position paper | Fairness, missing counters |
| Translate technical material | Public technical doc | Accuracy, nuance loss |
| Review source coverage | Small public corpus | Missing key sources |
| Current-awareness omission | Corpus with dated materials | Recency failure |
| Literature map comparison | Model map vs verified sources | Fabricated or missing citations |

## Do not use without authorization

- Unpublished interviews
- Confidential notes
- Privileged material
- Restricted datasets
- Identifiable participant information

Confirm institutional authorization and appropriate tools before using sensitive materials.

## Reference criteria

Establish before testing:

- What counts as accurate summary?
- Which elements must not be omitted (methods, limitations, dissent)?
- How will citations be verified?

## Does not prove

Research workflow success on public materials does not establish readiness for confidential scholarship, clinic work, or litigation support.
