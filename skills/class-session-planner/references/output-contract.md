# Output Contract

Support modular output. Do not generate every deliverable unless the user asks for a complete session package.

Deliverables: session design brief; detailed run of show; teaching notes; question sequence; activity instructions; board or slide plan; reading and preparation guide; formative check; contingency plan; post-class reflection; session design review.

File outputs may be editable Word lesson plan, Google Docs-compatible content, clean Markdown, plain text, student-facing handout, faculty-only notes, or separate reflection form. Do not claim a file was created unless the file-generation action succeeds.
