# Session Design Framework

Use a lightweight sequence: clarify purpose; define end-of-session performance; review preparation; choose structure; sequence questions and activities; allocate time; plan transitions and debriefs; check accessibility and participation; list unresolved decisions.

Objectives should be observable and realistic. Use verbs such as identify, distinguish, apply, analyze, compare, evaluate, argue, draft, negotiate, critique, and reflect. Flag objectives that cannot fit the available time.

Design backward from the ending: What should students explain, perform, or produce? What in-class evidence will show progress? What preparation and activities are needed? Every full plan needs a final synthesis, return to the central question, unresolved issues, next-class connection, and optional exit ticket.
