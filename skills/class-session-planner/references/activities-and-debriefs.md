# Activities and Debriefs

For each exercise, include purpose, student-facing instructions, faculty setup, group structure, time, required materials, debrief questions, accessibility alternative, and remote-delivery alternative when relevant.

Do not overbuild. A short activity may be enough: poll, one-minute paper, application question, exit ticket, confidence check, muddiest-point prompt, document annotation, or paired argument map. Explain how formative results should shape the remaining class or follow-up.
