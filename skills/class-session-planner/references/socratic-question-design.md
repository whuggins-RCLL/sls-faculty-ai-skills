# Socratic Question Design

Organize questions by instructional purpose rather than a flat list. Possible categories: orientation and procedural posture; facts that matter; rule identification; court reasoning; factual distinction; counterargument; policy consequences; application to a new scenario; synthesis across authorities; professional judgment; reflection.

For each major question, include branching follow-ups: if correct, ask for application or limitation; if incomplete, direct attention to a fact or passage; if mistaken, ask a diagnostic question before correcting. Do not design humiliating, manipulative, or trick cold calls. Consider advance notice, wait time, and multiple participation modes for complex questions.
