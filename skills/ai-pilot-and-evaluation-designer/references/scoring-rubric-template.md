# Scoring rubric template

Use anchored rubrics so different reviewers score the same output the same way. Define the rubric before scoring.

## Rubric structure

For each dimension that matters to the decision, define levels with concrete anchors.

| Dimension | Level 3 (meets) | Level 2 (partial) | Level 1 (fails) | Critical? |
|---|---|---|---|---|
| Source integrity | Every proposition traceable to a real, on-point source | Minor citation gaps, no fabrication | Any fabricated or non-existent authority | Yes |
| Completeness | No material omission | Minor omission | Material adverse authority or fact omitted | Sometimes |
| Accuracy of law | Correct statement of current law | Minor imprecision | Misstatement of current law | Yes |
| Usefulness | Advances the task materially | Marginal help | No help or net negative | No |
| Clarity/format | Clear, well-structured | Usable with edits | Confusing | No |

Adapt the dimensions to the task; do not keep dimensions that do not affect the decision.

## Anchoring rules

- Each level must describe observable features, not vibes.
- Mark which dimensions can trigger a critical failure regardless of other scores.
- Provide one worked example per dimension so reviewers calibrate.

## Aggregation caution

Report dimension scores separately. If you compute any overall score, it must not let strong dimensions cancel a critical failure. State the aggregation rule explicitly (for example, "any Critical = overall fail").

## Reviewer instructions

Include: what to read, how to record, what to do when uncertain, and how disagreements are resolved.
