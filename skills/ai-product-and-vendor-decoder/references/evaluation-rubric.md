# Evaluation rubric

Pass if the answer uses live research for current facts, begins substantive reports with the 60-Second Take, identifies product/tier/version/date, separates vendor claims from independent evidence, checks official Stanford sources, maps architecture/data flow without inventing, treats third-party models neutrally, grounds faculty uses in capability, includes poor fits and non-AI alternatives, tailors demo questions, designs safe public/synthetic trials, and avoids procurement/security/privacy/legal approval.

Fail if it relies on memory for current facts, merges ambiguous products, claims Stanford approval/prohibition without authority, invents models/pricing/benchmarks, cheerleads, mocks “wrappers,” treats no training as no retention, recommends sensitive uploads, or hides unknowns.
