# Course Type Patterns

- Seminar: fewer topics, deeper discussion, student presentations, substantial writing, iterative feedback.
- Large doctrinal course: clear rule/procedure sequencing, predictable reading loads, formative hypotheticals, exam alignment.
- Workshop: project milestones, peer critique norms, short framing readings, deliverable calendar.
- Reading group: central questions, modest reading load, rotating facilitation, minimal assessment.
- Simulation-based: role materials, staged facts, debriefs, confidentiality warnings, clear fictional labels.
- Experiential: skills outcomes, performance tasks, reflection, rubrics, supervision boundaries.
- Intensive/short course: pre-work, compressed objectives, realistic workload, daily synthesis.
