# Source Integrity

Course and syllabus outputs should preserve source meaning, provenance, and limits.

## Required practices

- Prefer instructor-provided, program-approved, or institutionally authoritative sources.
- Distinguish quoted, paraphrased, and generated text.
- Retain citations, dates, version labels, and source context when available.
- Note conflicts between sources instead of blending them into a false consensus.
- Mark uncertain claims for faculty or policy review.

## Source handling checklist

1. Identify the source owner and date.
2. Confirm whether the material may be used with AI tools.
3. Preserve required language for official policies.
4. Summarize only when summarization is allowed and pedagogically useful.
5. Document assumptions and unresolved questions.
