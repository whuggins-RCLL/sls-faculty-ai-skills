# Output Contract

A full modular course package contains:

1. Course design brief.
2. Learning objectives.
3. Course architecture.
4. Course calendar.
5. Updated/new syllabus.
6. Separate reading list.
7. AI policy as syllabus section and separate document.
8. Design review memo.
9. Revision log when revising an existing course.

Faculty may request selected components. Always include assumptions, unresolved decisions, source labels, and human-review warnings. Export as clean Markdown, plain text, Google Docs-compatible text, or Word-ready content; never claim file creation unless an action succeeds.
