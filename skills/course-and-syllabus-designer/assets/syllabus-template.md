# Syllabus Template Asset

This text asset is intentionally committed instead of a binary `.docx` file so repository diff and review tools can inspect every change.

Use `../references/syllabus-template.md` as the authoritative structure for syllabus generation. When a Word document is needed, generate it outside the committed source tree from the Markdown/Google Docs-compatible content and verify that the binary export action succeeded before telling a faculty member it exists.
